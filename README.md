Моделирует работу автоматизированной системы управления технологическим процессом. Подходит для демонстрации базовых принципов АСУ: сбор данных с датчиков, ручное и автоматическое управление механизмами, аварийная сигнализация.
- Ручное управление нагревателем и насосом через консольные команды
- Автоматическая проверка аварийных порогов с выводом предупреждений
- Динамическое обновление параметров в реальном времени
